# Getting started

This section guides you through installing Cedalion and running your first analysis.

**Recommended reading order:**

1. [Concepts](../concepts.md) — if you are new to fNIRS, start here to learn the measurement physics and Cedalion's core data model before writing any code.
2. [Installation](installation.md) — set up a conda environment and install the package.
3. [Google Colab setup](colab_setup.md) — run example notebooks in the cloud from your browser without a local installation.
4. [Quick start](quickstart.md) — load a recording, compute haemoglobin concentration, and plot a channel in under 20 lines.
5. [Tutorial notebooks](../tutorial.rst) — seven end-to-end notebooks covering the full pipeline from data loading to image reconstruction.
6. [Contributing code](contributing_code/contributing_code.md) — how to add new functions, follow the style guide, and submit a pull request.
